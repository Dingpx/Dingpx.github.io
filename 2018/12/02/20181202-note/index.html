列表

标记

  1. 有序列表：“ 数字 + . + 内容 ”
    1. 内容
  • 无序列表：“ +/-/* + 内容 ”
    • 内容
  • 任务列表：“ +/-/* + [ ] + 内容”
    • 内容

快捷键

  1. 有序列表：⌘ + ⎇ + O

  • 无序列表：⌘ + ⎇ + U

  • [ ] 任务列表：⌘ + ⎇ + X

  • 减少缩进：⌘ +「

    • 增加缩进： ⌘ + 」
